-----------WEBPACK--------------
- Pega os modulos da aplicação, e junta em um pacote só
-Gera o arquivo final em es5

---------INSTALAR WEBPACK--------

- package.json vazio
- npm install --save-dev webpack
- criar arquivo webpack.config.js
- configurar esse arquivo parar ler um arquivo e gerar um bandle (pacote)

- criar o arquivo src/index.js
- npm i -g webpack
- npm install -g webpack-cli -D
- Criar o index.html e chamar o bundle

-----------SERVIDOR WEBPACK-------
- npm i -g webpack-dev-server
- publicPath:'/static' no webpack.config
- mudar o caminho no index para static/bundle
- webpack-dev-server
