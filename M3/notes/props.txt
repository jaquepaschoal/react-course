---------- PROPS ----------------
- usado quando precisa passar algum atributo pro elemento, como id.

-this.props.propriedade = pega a props que vc colocou no elemento

------- ATRIBUTOS HTML PROPS----------

- Ao inves de usar 'class' deve usar 'className'
- Ao inves de usar 'for' usa 'htmlFor'

---------- GET DEFAULT PROPS-------

- Dá valores por padrão para as propriedades;
- getDefaultProps: function() {
    return {
      name:'Desconhecido'
    }
  },


---------- PASSANDO OUTROS DADOS VIA PROPS ---------------

- Para colocar um codigo como propriedade, basta utilizar chaves;
- Exemplo: lastname= {2-1}
